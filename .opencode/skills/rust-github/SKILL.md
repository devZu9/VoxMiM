# Rust — GitHub

- **Пуш на GitHub только после явного подтверждения пользователя.** Никогда не пушить без разрешения.
- Напомнить о пуше после каждых 5 изменённых файлов. Если изменений меньше 5, но пользователь говорит «готово», «заканчиваем», «на сегодня хватит» — тоже спросить про пуш.
- Токен: `%USERPROFILE%\.github_token` (ограничен по времени).
- ✅ Для публикации релизов с кириллицей — `gh` CLI через `cmd.exe`, тело в файле UTF-8 без BOM, флаг `--notes-file`
- ❌ НЕ использовать PowerShell `Invoke-RestMethod` — он кодирует тело в Windows-1251, кириллица в release notes превращается в кракозябры
