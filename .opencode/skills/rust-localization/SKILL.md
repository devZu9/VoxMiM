# Rust — Многоязычность

- Проект должен быть готов к локализации с самого начала.
- Сразу на стадии формирования архитектуры предусмотреть папку `lang/` с JSON-файлами локалей (путь до папки получить через `dirs` крейт или API фреймворка — загрузи и используй `rust-single-source`).
- При проектировании новых строк учитывать склонения языковых групп:
  - одно яйцо, два яйца, много яиц.
  - работать с настройкой, взять из настроек, сохранить в настройках.
