# Rust — Размер файлов

- **Максимум по умолчанию:** 250 строк на файл.
- **Допустимое превышение:** до 50% (до 375 строк), если:
  - контекст файла един и целостен,
  - код не подразумевает переиспользование из других мест,
  - и при этом **не** создаются вспомогательные файлы.
- Если для файла понадобились вспомогательные файлы — правило +50% **не действует**: все файлы группы (основной + хелперы) ≤250 строк каждый.

## Механизм при превышении 375 строк

Если файл превышает 375 строк, но контекст един:

1. **Шаг 1 — хелперы:** вынести чистые вспомогательные функции в `{context}_helpers.rs` (например, `tab_status_helpers.rs`). После этого **все** файлы группы ≤250 строк.

2. **Шаг 2 — директория:** если после Шага 1 файл всё ещё >250 строк, превратить в директорию `{context}/`:
   - `{context}/mod.rs` — публичное API и реэкспорты
   - `{context}/render.rs` — отрисовка UI
   - `{context}/handlers.rs` — обработчики событий
   - `{context}/helpers.rs` — внутренние утилиты
   - Каждый файл в директории — ≤250 строк.

3. **Шаг 3 — переиспользование:** если хелперы понадобились извне — переместить на уровень выше (в модуль родителя), в `{context}/` оставить только публичный реэкспорт.
