# Rust — Автосохранение настроек

- Все имеющиеся и будущие настройки должны автосохраняться при изменении (`save_setting`).
- Автовосстанавливаться после загрузки/перезагрузки проекта.
- Файл настроек: в папке data проекта (путь получить через `dirs` крейт или API по фреймворку — загрузи и используй `rust-single-source`).
